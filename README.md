# robot-vla-video

基于 **真实人类演示视频** 的 **人形机器人 VLA（Vision-Language-Action）动作模仿与意图理解** 项目。  
项目目标是在 **Isaac Sim** 中构建一个完整系统，实现：

1. 从真实人类视频中提取人体关键点与动作信息  
2. 将视频动作映射为 humanoid 机器人的关节动作序列  
3. 学习动作背后的任务意图（Intent）  
4. 在仿真环境中生成并执行可泛化的机器人动作策略  

---

# 1. 项目背景

传统模仿学习常依赖：
- 动作捕捉设备
- 遥操作示教
- 仿真环境中人工生成轨迹

这些方法虽然精度较高，但成本高、扩展性差。  
相比之下，**真实人类视频**具有以下优势：

- 数据获取成本低，规模大
- 动作自然、细节丰富
- 包含丰富场景上下文
- 更适合学习复杂任务意图
- 可支持更强的泛化能力

本项目希望结合：
- **Video-to-Motion**：从视频到机器人动作
- **Intent-conditioned Policy**：基于意图的动作生成
- **Isaac Sim 高精度仿真**
- **Humanoid Robot 控制**

构建一个具备科研价值的完整研究管线。

---

# 2. 项目目标

本项目分为两个核心研究方向：

## Idea 1：视频驱动的精细动作模仿
输入真实人类视频，输出机器人可执行的关节动作序列。

目标：
- 提取人体关键点
- 建立视频时序表示
- 映射到 humanoid 机器人关节空间
- 在仿真环境中执行并评估

## Idea 2：动作意图理解与自主执行
不仅模仿动作，还理解动作背后的意图，并在变化场景中自主完成任务。

目标：
- 从动作序列中学习意图表征
- 构建意图条件策略网络
- 在不同场景中完成动作泛化

---

# 3. 项目整体架构

完整流程如下：

```text
真实人类视频
   ↓
关键点提取（MediaPipe / OpenPose）
   ↓
时序动作编码（Video Encoder / Transformer）
   ↓
机器人动作生成（Video2Motion）
   ↓
动作意图编码（Intent Encoder）
   ↓
意图条件策略（Policy Network）
   ↓
Isaac Sim 中 humanoid 机器人执行
   ↓
动作模仿评估 + 任务成功率评估
